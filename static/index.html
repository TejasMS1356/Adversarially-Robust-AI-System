How It Defends

🧹
Data Sanitization
Z-score filtering and spectral signature analysis removes poisoned training samples before the model ever learns from them.
⚔️
Adversarial Training
PGD-3 attack examples are generated every batch. The model trains on 50% clean + 50% adversarial inputs, building robust decision boundaries.
🔍
Input Detection
Feature squeezing with 5 squeezers detects adversarial inputs at runtime by checking if predictions change after bit-depth reduction and smoothing.
🛡️
Robust Architecture
ResNet-style CNN with residual blocks, batch normalization and dropout trained for 60 epochs on CIFAR-10 with cosine annealing LR.
